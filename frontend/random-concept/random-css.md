  # Grid Layout and Spacing

In **Grid**, child elements usually don’t need fixed width/height because Grid automatically distributes space.

## Creating Space Between Elements

To create space between elements (like `h1` and `p`), use:
- **gap** (preferred)
- or **margin**

## Vertical Layouts

For vertical layouts, use:
- `flex flex-col` → stacks elements
- `gap-*` → adds space between them

## Creating Space Inside Text (Between Lines)

Use:
- **line-height** → `leading-*` in Tailwind

## Key Understanding:
- **gap / margin** → controls space between elements
- **line-height (leading)** → controls space between lines of text
